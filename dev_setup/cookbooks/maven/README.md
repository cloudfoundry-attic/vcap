Description
===========

Installs and configures Apache Maven

Requirements
============

Platform: 

* Debian, Ubuntu, CentOS, Red Hat, Fedora

The following Opscode cookbooks are dependencies:

* java
* jpackage

Attributes
==========

Usage
=====

Simply include the recipe where you want Apache Maven installed.
